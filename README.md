# krewdev/homebrew-tap

Homebrew formulas for:

- [privacy-kit](https://github.com/krewdev/privacy-kit) (`pk`)
- [hubstaff-work-shell](https://github.com/krewdev/hubstaff-work-shell) (`hs`)

## Install

```bash
brew tap krewdev/tap
brew install privacy-kit
brew install hubstaff-work-shell
```

## Update formulas

When tagging a new release, update `url` / `sha256` in `Formula/*.rb` and open a PR to this tap.
