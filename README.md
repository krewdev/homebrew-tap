# krewdev/homebrew-tap

Homebrew formulas for:

| Formula | Binary | Project |
|---------|--------|---------|
| `privacy-kit` | `pk` | [privacy-kit](https://github.com/krewdev/privacy-kit) |
| `hubstaff-work-shell` | `hs` | [hubstaff-work-shell](https://github.com/krewdev/hubstaff-work-shell) |

## Install

```bash
brew tap krewdev/tap
# Homebrew 6+ may require trusting third-party taps once:
brew trust krewdev/tap   # if prompted / refused as untrusted

brew install privacy-kit
brew install hubstaff-work-shell

pk audit
hs doctor
```

## Versions

- privacy-kit: **v0.2.1**
- hubstaff-work-shell: **v0.1.2**
